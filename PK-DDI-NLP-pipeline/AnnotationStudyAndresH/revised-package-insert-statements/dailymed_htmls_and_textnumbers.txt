How to read the information: After each URL, the numbers of the corresponding text files are listed, then comes the type of drug insert the file comes from. For "new" ones, the first file is the Drug Interactions section of the drug insert and will be found in the drug_interaction_sections folder, and the second file is the Clinical Pharmacology section of the drug insert and will be found in the clinical_pharm_section folder. For ones that say "one random", the text files come from old inserts, with the first files being interaction statements (found in the interaction_statements_from_old folder) and the last file being the random paragraph (the random paragraph file number will ALWAYS be the last number of the set), which are found in the random_statements folder.

Examples: (5-6, new) = first file (5) will be Drug Interaction section, second file (6) will be Clinical Pharmacology section
	(11-20, one random) = files 11-19 will be interaction statements and the last file (20) is the random paragraph
